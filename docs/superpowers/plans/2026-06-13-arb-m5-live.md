# M5 — Live Execution Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the M4 go-live stub with a real Polymarket CLOB trading path — EIP-712 signing (signature_type 1 / Magic proxy), L1/L2 auth, FAK-only `LiveVenue`, pure-buy-only live dispatch behind `--live` + typed confirmation, $10/basket + $25 session-loss canary caps — staged recon → safety pre-reqs → signing → auth → venue → wiring → shadow → funded round-trip.

**Architecture:** `LiveVenue` implements the existing `ExecutionVenue` trait so the coordinator/executor/store/TUI keep their shapes. New pure modules `sign.rs` (EIP-712) and `auth.rs` (L1 derive + L2 HMAC headers) in `pm-execution`, secrets from env only. Live-mode gates live in the coordinator (pure-buy filter, canary caps, latched `SessionLoss` halt). Safety pre-reqs flagged in M3/M4 code (capped mid-mark, ws heartbeat, per-feed staleness) land before any live code.

**Tech Stack:** Rust, tokio, reqwest (already in workspace); NEW workspace deps: `alloy-primitives`, `alloy-sol-types`, `alloy-signer`, `alloy-signer-local` (EIP-712), `hmac`, `sha2`, `base64` (L2 auth).

```rust
/// An owned secret string. Debug prints `Secret(<redacted>)`; no Display.
pub struct Secret(String);

impl Secret {
    pub fn new(s: String) -> Self {
        Secret(s)
    }
    /// The raw value. Callers must never log it.
    pub fn expose(&self) -> &str {
        &self.0
    }
    /// Hex private key with any `0x` prefix stripped.
    pub fn expose_key_hex(&self) -> String {
        self.0.strip_prefix("0x").unwrap_or(&self.0).to_string()
    }
}

impl std::fmt::Debug for Secret {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        write!(f, "Secret(<redacted>)")
    }
}

/// Pre-derived L2 API credentials (all three or none).
#[derive(Debug)]
pub struct ApiCreds {
    pub key: String,
    pub secret: Secret,
    pub passphrase: Secret,
}

/// Everything live mode reads from the environment.
#[derive(Debug)]
pub struct LiveSecrets {
    pub private_key: Secret,
    /// Magic/email accounts: the Polymarket proxy wallet (maker). Optional —
    /// when absent it is looked up per RECON-M5.md at startup.
    pub proxy_address: Option<String>,
    /// When present, API-key derivation is skipped.
    pub api: Option<ApiCreds>,
}

impl LiveSecrets {
    /// Production entrypoint.
    pub fn from_env() -> Result<Self, String> {
        Self::from_lookup(|k| std::env::var(k).ok())
    }

    /// Testable core: `lookup` returns the env value for a name.
    pub fn from_lookup(lookup: impl Fn(&str) -> Option<String>) -> Result<Self, String> {
        let private_key = Secret::new(
            lookup("PM_PRIVATE_KEY").ok_or("PM_PRIVATE_KEY not set (export your wallet key from Polymarket settings)")?,
        );
        let proxy_address = lookup("PM_PROXY_ADDRESS");
        let api = match (lookup("PM_API_KEY"), lookup("PM_API_SECRET"), lookup("PM_API_PASSPHRASE")) {
            (None, None, None) => None,
            (Some(key), Some(secret), Some(pass)) => Some(ApiCreds {
                key,
                secret: Secret::new(secret),
                passphrase: Secret::new(pass),
            }),
            (k, s, p) => {
                let mut missing = Vec::new();
                if k.is_none() {
                    missing.push("PM_API_KEY");
                }
                if s.is_none() {
                    missing.push("PM_API_SECRET");
                }
                if p.is_none() {
                    missing.push("PM_API_PASSPHRASE");
                }
                return Err(format!("partial PM_API_* credentials; missing: {}", missing.join(", ")));
            }
        };
        Ok(LiveSecrets {
            private_key,
            proxy_address,
            api,
        })
    }
}
```

```rust
/// Sign `order` for the regular or neg-risk exchange. Returns 0x-hex (65-byte
/// r||s||v) exactly as the venue expects.
pub fn sign_order(
    signer: &PrivateKeySigner,
    order: &ClobOrder,
    neg_risk: bool,
) -> Result<String, SignError> {
    let token: U256 = order
        .token_id
        .parse()
        .map_err(|_| SignError::BadTokenId(order.token_id.clone()))?;
    let sol_order = Order {
        salt: U256::from(order.salt),
        maker: order.maker,
        signer: order.signer,
        taker: order.taker,
        tokenId: token,
        makerAmount: U256::from(order.maker_amount),
        takerAmount: U256::from(order.taker_amount),
        expiration: U256::from(order.expiration),
        nonce: U256::from(order.nonce),
        feeRateBps: U256::from(order.fee_rate_bps),
        side: match order.side {
            Side::Buy => 0,
            Side::Sell => 1,
        },
        signatureType: order.signature_type,
    };
    let hash = sol_order.eip712_signing_hash(&domain(neg_risk));
    let sig = signer
        .sign_hash_sync(&hash)
        .map_err(|e| SignError::Signer(e.to_string()))?;

}

/// (makerAmount, takerAmount) for a leg, µ units, rounded against us.
/// BUY: maker = µUSDC outlay (cost rounded up), taker = µshares.

pub fn clob_amounts(action: Action, ts: TickSize, limit_px: Px, qty: Qty) -> (u64, u64) {

        Action::Buy => {

        Action::Sell => {
            let proceeds = pm_core::cash::sell_proceeds(px_micro, qty);
            (qty.0, proceeds.0.max(0) as u64)
        }
    }
}
```

```rust
use alloy_signer_local::PrivateKeySigner;
```

```rust
use base64::Engine;
use base64::engine::general_purpose::URL_SAFE;

/// RECON-pinned attestation message (RECON-M5.md).
const CLOB_AUTH_MESSAGE: &str = "This message attests that I control the given wallet";

sol! {
    struct ClobAuth {
        address address;
        string timestamp;
        uint256 nonce;
        string message;
    }
}

#[derive(Debug)]
pub enum AuthError {
    Sign(String),
    BadSecret(String),
    Http(String),
}

impl std::fmt::Display for AuthError {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            AuthError::Sign(e) => write!(f, "L1 signing: {e}"),
            AuthError::BadSecret(e) => write!(f, "API secret not base64url: {e}"),
            AuthError::Http(e) => write!(f, "auth http: {e}"),
        }
    }
}
impl std::error::Error for AuthError {}

/// L1: sign the ClobAuth attestation (domain: ClobAuthDomain v1, chain 137,
/// NO verifying contract). Returns 0x-hex signature.
pub fn l1_signature(
    signer: &PrivateKeySigner,
    timestamp: &str,
    nonce: u64,
) -> Result<String, AuthError> {
    let msg = ClobAuth {
        address: signer.address(),
        timestamp: timestamp.to_string(),
        nonce: alloy_primitives::U256::from(nonce),
        message: CLOB_AUTH_MESSAGE.to_string(),
    };
    let domain = eip712_domain! {
        name: "ClobAuthDomain",
        version: "1",
        chain_id: super::sign::CHAIN_ID,
    };
    let hash = msg.eip712_signing_hash(&domain);
    let sig = signer
        .sign_hash_sync(&hash)
        .map_err(|e| AuthError::Sign(e.to_string()))?;
    Ok(format!("0x{}", alloy_primitives::hex::encode(sig.as_bytes())))
}

/// L2: base64url(HMAC-SHA256(base64url-decode(secret), ts+METHOD+path+body)).
pub fn l2_signature(
    secret_b64url: &str,
    timestamp: &str,
    method: &str,
    path: &str,
    body: Option<&str>,
) -> Result<String, AuthError> {
    let key = URL_SAFE
        .decode(secret_b64url)
        .map_err(|e| AuthError::BadSecret(e.to_string()))?;
    let mut mac =
        Hmac::<Sha256>::new_from_slice(&key).map_err(|e| AuthError::BadSecret(e.to_string()))?;
    mac.update(timestamp.as_bytes());
    mac.update(method.as_bytes());
    mac.update(path.as_bytes());
    if let Some(b) = body {
        mac.update(b.as_bytes());
    }
    Ok(URL_SAFE.encode(mac.finalize().into_bytes()))
}

/// The five L2 headers for a trading request, in (name, value) pairs.
pub fn l2_headers(
    creds: &ApiCreds,
    eoa_address: &str,
    timestamp: &str,
    method: &str,
    path: &str,
    body: Option<&str>,
) -> Result<Vec<(&'static str, String)>, AuthError> {
    let sig = l2_signature(creds.secret.expose(), timestamp, method, path, body)?;
    Ok(vec![
        ("POLY_ADDRESS", eoa_address.to_string()),
        ("POLY_SIGNATURE", sig),
        ("POLY_TIMESTAMP", timestamp.to_string()),
        ("POLY_API_KEY", creds.key.clone()),
        ("POLY_PASSPHRASE", creds.passphrase.expose().to_string()),
    ])
}

/// Derive (GET /auth/derive-api-key) or create (POST /auth/api-key) the API
/// key with L1 headers. `base` e.g. https://clob.polymarket.com. Uses the
/// venue server time (seconds) for the timestamp.
pub async fn derive_or_create_api_key(
    http: &reqwest::Client,
    base: &str,
    signer: &PrivateKeySigner,
    server_time_s: u64,
) -> Result<ApiCreds, AuthError> {
    let ts = server_time_s.to_string();
    let sig = l1_signature(signer, &ts, 0)?;
    let l1 = [
        ("POLY_ADDRESS", format!("{}", signer.address())),
        ("POLY_SIGNATURE", sig),
        ("POLY_TIMESTAMP", ts.clone()),
        ("POLY_NONCE", "0".to_string()),
    ];
    for (method, path) in [("GET", "/auth/derive-api-key"), ("POST", "/auth/api-key")] {
        let url = format!("{base}{path}");
        let mut req = if method == "GET" {
            http.get(&url)
        } else {
            http.post(&url)
        };
        for (k, v) in &l1 {
            req = req.header(*k, v);
        }
        let resp = req.send().await.map_err(|e| AuthError::Http(e.to_string()))?;
        if resp.status().is_success() {
            let v: serde_json::Value = resp
                .json()
                .await
                .map_err(|e| AuthError::Http(e.to_string()))?;
            let get = |k: &str| v[k].as_str().map(str::to_string);
            if let (Some(key), Some(secret), Some(pass)) =
                (get("apiKey"), get("secret"), get("passphrase"))
            {
                return Ok(ApiCreds {
                    key,
                    secret: crate::secrets::Secret::new(secret),
                    passphrase: crate::secrets::Secret::new(pass),
                });
            }
        }
    }
    Err(AuthError::Http("derive and create both failed".into()))
}
```
